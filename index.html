Quality of Life Data Exploration

Welcome to our Project page!

We've taken data from the Teleport API (view it on the Data and Methodology page) to compare 256 cities and explore trends in quality of life.

Our site is divided into three main sections: Charts by Indicator, Charts by City, and Data/Methodology.

Charts by Indicator contains two sets of charts that visualize the indicator data we pulled and the user can select which city they'd like to look at.

Charts by City contains two sets of charts that provide a birds eye view of the scores for all cities. The user can select which indicator they'd like to look at.

Data and Methodology contains the data we pulled from the Teleport API. The user can download the data and explore it themselves.

Key Takeaways From Data:

  • The United States isn't the absolute best place to live - none of the top 20 cities in our data are situated in the US.
  • The top cities to live in include Singapore, Munich, and Toronto.
  • In general, the scores tend to favor cities in "developed" countries.
